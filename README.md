### Для запуска:
+ Развернуть виртуальное окружение, установить зависимости и запустить main.py или main1.py:
```
python -m venv venv 
source venv/Scripts/activate
pip install -r requirements.txt
```

+ Для запуска тестов:
```
pytest
```
